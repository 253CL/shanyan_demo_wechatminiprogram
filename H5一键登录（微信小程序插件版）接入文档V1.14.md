一键登录插件可以让用户快速进行一键登录，提供标准授权页以及直接获取用户token的登录方式。

2) 流程说明

1. 业务方小程序申请接入取号插件，从业务方服务端获取 md5 签名

2. 业务方 H5 携带签名、wxappid 等参数，调用一键登录的取号方法请求取号

3. 取号插件自动进行网络状态判断，根据判断结果进行下一步流程：

+ 若为wifi/热点/无网络：取号插件返回网络状态结果，并结束取号；业务方根据返回结果自行处理
+ 若为数据/未知网络：取号插件向认证服务端发起取号请求，取号成功后认证服务端返回临时凭证、手机号掩码，插件重定向展示授权页面

4. 用户点击确认授权或补齐正确的手机号码，授权页面发起请求获取 token，认证服务端返回token 并将 token 返回业务方小程序

5. 业务方小程序携带 token 发起 token 校验请求，认证服务端校验通过后返回加密手机号

# 7 接入指引
## 7.1 插件申请
插件申请在腾讯公众平台中， “微信小程序官方后台-设置-第三方服务-插件管理” 里点击 “添加插件”，搜索 “号码认证” 申请，审核通过后，小程序开发者可在小程序内使用该插件。

# 附录2：常见问题
**1. 是否支持wifi环境下使用？**

答：仅支持数据流量下使用，不支持wifi、热点环境下使用。请提醒用户请勿在热点环境下使用。

**2. 授权页输入错误号码是否有限制次数？**

答：同一号码连续输入3次错误，将会被锁定。为避免此问题，请开发者注意在联调过程中请勿连续输错3次。

**3. jssdk和token校验接口中的sign是否相同？**

答：不相同。两者的生成工具也不同，生成jssdk中的sign请使用附录1中MD5签名工具类；生成token校验接口的sign请使用附录2中“SHA256withRSA签名方法”。
