# H5版初始化

本文介绍了闪验服务的默认POST初始化请求方式。用户需要发送POST请求到/sy/h5/init路径，并提供电信类型、应用ID和数据等参数。成功的请求将返回包含应用ID、跟踪ID、时间戳、版本、签名和应用密钥等信息的JSON响应。返回状态码为0，表示请求成功。

释义如下：

|cmccTimestamp|cuccTimestamp|ctccTimestamp|
|---|---|---|
|*移动 时间戳*|*联通 时间戳*|*电信 时间戳*|

# **闪验**

v1\.0\.0

Base URLs:

- [开发环境: ](http://127.0.0.1:18080/)[https://sy\.cl2m\.cn/sy/h5/init](http://127.0.0.1:18080/)

# **Default**

## **POST 初始化**

POST /sy/h5/init

> Content\-type  form\-data
> 
> 



> Body 请求参数示例
> 
> 

```YAML
appId: tqmXaqsR
data: string
```
